Blog Source

该git库用于存储个人博客公共source部分文件

以下说明在不同PC端pull该source之后需要安装的其他插件，当然前提是必须已搭建好hexo等博客必备的东西。

hexo版本4.0.0或以上，nodejs版本最新即可

1 主题

该博客使用主题hexo-theme-butterfly

  • 主题安装

    git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/Butterfly

  • 主题需要pug以及stylus渲染器

    npm install hexo-renderer-jade hexo-renderer-stylus –save

    or

    yarn add hexo-renderer-jade hexo-renderer-stylus

2 搜索系统

目前该博客使用本地搜索

  • 本地搜索

    npm install hexo-generator-search –save

  • Algolia

    如果使用Algolia搜索系统，则需要安装 hexo-algolia 或者 hexo-algoliasearch

3 文章置顶

文章置顶需要安装相关插件

npm uninstall hexo-generator-index –save

然後

npm install hexo-generator-index-pin-top –save

4 添加宠萌

博客添加了宠萌

  • 先安装插件：npm install hexo-helper-live2d –save
  • 选择宠萌： npm install live2d-widget-model-wanko

5 添加音乐歌单页面

博客使用了hexo-tag-aplayer插件

npm install –save hexo-tag-aplayer

6 页面添加视频

博客留言板中插入了视频，插入视频需要插件：hexo-tag-dplayer

npm install –save hexo-tag-dplayer

7 博客加入豆瓣功能

博客在清单项加入了豆瓣功能（电影、图书、游戏），加入豆瓣功能使用插件：hexo-douban

npm install hexo-douban –save

启动及发布部署时注意：

hexo d : hexo douban 和 hexo deploy 缩写都是hexo d,故在发布部署是使用全名

本地启动：

hexo clean && hexo douban -bgm && hexo g && hexo s

发布部署：

hexo clean && hexo douban -bgm && hexo g && hexo deploy

当然也可以修改_config.yml中douban属性builtin为true,意为将豆瓣页面生成功能嵌入到hexo g和hexo s中

8 博客跳转其他页面

博客跳转自动到其他页面如：博客中的购房预算:

hexo new page “对应页面”，修改对应页面下的index.md,如购房预算（hbudget）,layout为false即不渲染主题，不同页面修改url地址即可
